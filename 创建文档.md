## 第二步：设计数据库 (models.py)

我们要设计一张表来存传感器数据。 打开 `device_api/` 文件夹下的 `models.py` 文件。 把它原本的内容清空，复制粘贴下面的代码：

```python
from django.db import models+


# Create your models here.

# 想象我们在设计一张 Excel 表格，表名叫 SensorData
class SensorData(models.Model):
    # 第一列：温度 (浮点数，比如 25.5)
    temperature = models.FloatField()
    
    # 第二列：湿度 (浮点数，比如 60.2)
    humidity = models.FloatField()
    
    # 第三列：时间 (自动记录当前时间，不需要手动填)
    timestamp = models.DateTimeField(auto_now_add=True)

    # 这是一个方便给人类看的功能
    # 当你在后台查看数据时，它会显示 "时间 - 温度" 这样的格式，而不是 "Object(1)"
    def __str__(self):
        return f"{self.timestamp} - Temp: {self.temperature}"
```

## 第一步：编写业务逻辑 (views.py)

`views.py` 负责接收，处理后再返回结果。

打开 `device_api/views.py`，**清空**里面的内容，粘贴以下代码：

```python
from django.http import JsonResponse
from django.views.decorators.csrf import csrf_exempt
import json
from .models import SensorData

# ---------------------------------------------------------
# 接口 1: 接收数据 (专门给 ESP32 用)
# ---------------------------------------------------------
# @csrf_exempt 的意思是：允许“陌生人”发请求。
# 因为 ESP32 只是个芯片，没有浏览器的安全证书，不加这个它发不进来。
@csrf_exempt 
def receive_data(request):
    if request.method == 'POST':
        try:
            # 1. 拆开 ESP32 发来的包裹 (JSON数据)
            data = json.loads(request.body)
            
            # 2. 把数据写入刚才设计的数据库表格中
            SensorData.objects.create(
                temperature=data['temperature'],
                humidity=data['humidity']
            )
            
            # 3. 回复 ESP32："收到了！"
            return JsonResponse({"status": "success", "message": "Data saved!"})
            
        except Exception as e:
            # 如果出错 (比如发来的数据格式不对)，报错给它看
            return JsonResponse({"status": "error", "message": str(e)})
            
    return JsonResponse({"status": "error", "message": "Only POST allowed"})

# ---------------------------------------------------------
# 接口 2: 获取历史数据 (给 React 网页用)
# ---------------------------------------------------------
def get_history(request):
    # 1. 去数据库捞数据：按时间倒序(最新的在前面)，只取最近10条
    raw_data = SensorData.objects.all().order_by('-timestamp')[:10]
    
    # 2. 整理数据：把数据库对象转换成简单的字典列表
    result_list = []
    for item in raw_data:
        result_list.append({
            "temp": item.temperature,
            "humi": item.humidity,
            "time": item.timestamp.strftime("%H:%M:%S") # 把时间转成 "12:30:05" 这种格式
        })
    
    # 3. 把整理好的列表发回给网页
    return JsonResponse({"data": result_list})
```

## 第一步：创建模拟脚本

请在 `IoT_System` 文件夹下（也就是能看到 `manage.py` 的那个地方），新建一个文件，命名为 `simulate_device.py`。

**注意：这个文件是独立的，它不属于 Django，它只是一个测试工具。**

复制以下代码进去（已修复 URL 格式问题）：

```python
import requests
import random
import time

# 修正：去掉了多余的 markdown 符号，只保留纯净的网址
url = "http://127.0.0.1:8000/api/upload/"
print("开始模拟设备发送数据... (按 Ctrl+C 停止)")

while True:
    # 1. 生成随机的温度和湿度
    # random.uniform(20, 30) 意思是产生一个 20 到 30 之间的随机小数
    temp = round(random.uniform(20.0, 35.0), 1)
    humi = round(random.uniform(40.0, 60.0), 1)
    
    # 2. 打包数据 (JSON 格式)
    data = {
        "temperature": temp,
        "humidity": humi
    }
    
    try:
        # 3. 发送给服务器 (这行代码等同于 ESP32 发请求)
        response = requests.post(url, json=data)
        
        # 4. 打印结果
        if response.status_code == 200:
            print(f"✅ 发送成功: 温度={temp}°C, 湿度={humi}% | 服务器回复: {response.json()['message']}")
        else:
            print(f"❌ 发送失败: {response.status_code}")
            
    except Exception as e:
        print(f"❌ 连接错误: {e}")
        print("请检查：1. Django 服务器开启了吗？ 2. 地址对不对？")
        break
        
    # 5. 休息 2 秒再发下一次
    time.sleep(2)
```

## 第三步：编写网页代码 (App.jsx)

现在我们来写代码。请在编辑器中打开 `IoT_System/frontend/src/App.jsx` 文件。 **清空**里面的所有内容，复制粘贴下面的代码：

```python
import { useState, useEffect } from 'react'
import axios from 'axios'
import { LineChart, Line, XAxis, YAxis, CartesianGrid, Tooltip, Legend, ResponsiveContainer } from 'recharts'
import './App.css'

function App() {
  const [data, setData] = useState([])
  const [currentTemp, setCurrentTemp] = useState(0)
  const [currentHumi, setCurrentHumi] = useState(0)

  // 这个函数负责去后端拿数据
  const fetchData = () => {
    axios.get('http://127.0.0.1:8000/api/history/')
      .then(response => {
        const historyData = response.data.data
        // 如果拿到了数据
        if (historyData.length > 0) {
          // 1. 更新图表数据 (把原来的倒序 array 翻转一下，变成正序，方便画图从左到右)
          setData([...historyData].reverse())
          
          // 2. 更新当前显示的最新数值 (取第一个，因为后端发来的是最新的在前面)
          setCurrentTemp(historyData[0].temp)
          setCurrentHumi(historyData[0].humi)
        }
      })
      .catch(error => {
        console.error("哎呀，获取数据失败了:", error)
      })
  }

  // useEffect 相当于 "网页加载完毕后自动执行"
  useEffect(() => {
    // 1. 刚打开网页时，先拿一次数据
    fetchData()

    // 2. 之后每隔 2 秒钟自动去刷新一次数据 (轮询)
    const interval = setInterval(fetchData, 2000)

    // 清理工作 (当网页关闭时停止刷新)
    return () => clearInterval(interval)
  }, [])

  return (
    <div style={{ padding: '20px', fontFamily: 'Arial, sans-serif' }}>
      <h1>🌡️ 物联网实时监控系统</h1>
      
      {/* 实时数据显示卡片 */}
      <div style={{ display: 'flex', gap: '20px', marginBottom: '30px' }}>
        <div style={cardStyle}>
          <h3>当前温度</h3>
          <p style={{ fontSize: '40px', color: '#ff7300', margin: '10px 0' }}>{currentTemp}°C</p>
        </div>
        <div style={cardStyle}>
          <h3>当前湿度</h3>
          <p style={{ fontSize: '40px', color: '#387908', margin: '10px 0' }}>{currentHumi}%</p>
        </div>
      </div>

      {/* 历史趋势图 */}
      <div style={{ width: '100%', height: 400, background: '#f5f5f5', borderRadius: '10px', padding: '20px' }}>
        <h3>📊 历史趋势图</h3>
        <ResponsiveContainer width="100%" height="100%">
          <LineChart data={data}>
            <CartesianGrid strokeDasharray="3 3" />
            <XAxis dataKey="time" />
            <YAxis />
            <Tooltip />
            <Legend />
            <Line type="monotone" dataKey="temp" stroke="#ff7300" name="温度 (°C)" strokeWidth={3} />
            <Line type="monotone" dataKey="humi" stroke="#387908" name="湿度 (%)" strokeWidth={3} />
          </LineChart>
        </ResponsiveContainer>
      </div>
    </div>
  )
}

// 一些简单的样式配置
const cardStyle = {
  border: '1px solid #ddd',
  borderRadius: '8px',
  padding: '20px',
  width: '200px',
  textAlign: 'center',
  boxShadow: '0 2px 5px rgba(0,0,0,0.1)'
}

export default App
```
